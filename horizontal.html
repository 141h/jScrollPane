jScrollPane - horizontal demo page

This demonstration shows how jScrollPane works in a situation where the content of the scrollpane requires horizontal scrolling (e.g. it is wider than the space that is meant to contain it). The div below contains a very long unbroken text string which forces horizontal scrollbars when the overflow is set to "auto". jScrollPane recognises this and creates it's own horizontal scrollbars.
