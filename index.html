jScrollPane - cross browser styleable scrollbars with jQuery and CSS

This is the home of my jScrollPane plugin which provides the ability to convert a browsers default scrollbars (as seen on elements with overflow: auto or scroll) into pretty scrollbars which can be easily skinned using CSS.

The content of this page is still being written, for now I'm just using it to organise the links to the various demo/ test pages which I'm writing as I rewrite jScrollPane.

Simple examples/ tests

These examples show very basic functionality and exist so that I can test that any changes to jScrollPane work cross browser and don't cause any new problems.

  • Basic vertical scrolling
  • No scrollbars created when the content is too short to scroll
  • Vertical scrollbars can also appear on the left hand side of content
  • Basic horizontal scrolling
