jScrollPane - horizontal with arrows demo page

This demonstration enhances the basic horizontal demo and shows how you can pass showArrows: true in the settings to add arrows to the generated scrollbars.

The first div below contains a very long unbroken text string which forces horizontal scrollbars when the overflow is set to "auto". jScrollPane recognises this and creates it's own horizontal scrollbars. Note that the behaviour is the same as native scrollbars - only the one overflowing element flows outside the original width, the rest of the content is contained within it.

The second div below here contains an element whose own width is pushing the content out. Note that the scrollbar works as expected and all of the content flows within the 940px specifed as the width of the child paragraph.
